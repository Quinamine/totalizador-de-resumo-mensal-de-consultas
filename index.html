Sobre o totalizador

O Totalizador de Resumo Mensal de Consultas é um serviço online gratuito que, como o nome sugere, auxilia na elaboração do resumo mensal de consultas externas através do cálculo automático dos subtotais e totais a partir dos dados preenchidos pelo usuário (Profissional de Saúde). Tem a estrutura idêntica ao modelo da ficha de resumo mensal de consultas externas actualmente (2023) vigente no Serviço Nacional de Saúde (SNS) em Moçambique.

Qual é o propósito?

O propósito é de flexibilizar a elaboração do resumo auxiliando no cálculo dos subtotais e totais automaticamente e de minimizar possíveis erros de soma por confusão (por exemplo, somar valores de células não correspondentes, como feminino e masculino para achar um subtotal feminino ou vice versa) através do uso de um algorítimo atenciosamente programado para realizar cálculos de soma de forma correcta e com a vantagem de autopreencher as células correspondentes (subtotais e totais) .

Como funciona?

O serviço espera como parâmetros os dados colhidos do(s) Livro(s) de Registo de Consultas Externas da respectiva Unidade Sanitária durante o mês em consideração e com base nesses dados, as células correspondentes aos subtotais e totais são calculadas automaticamente e em tempo real. Após obter os totais, o usuário pode copiar os valores para a ficha original a partir da tela do seu dispositivo ou pode optar por imprimir a ficha do Totalizador para copiar a partir do papel.

O que o usuário precisa dispor?

Além de um dispositivo como computador (recomendável para uma melhor experiência) ou celular (alternativo) e outros serviços (internet) para ter acesso à plataforma, é importante que o profissional (usuário) tenha colhido os dados relativos as variáveis ou indicadores da ficha no(s) Livro(s) de Registo de Consultas Externas da sua Unidade Sanitária. Essa responsabilidade é inteiramente do usuário, a plataforma só se limita a fazer o cálculo dos subtotais e totais.

Qual é o destino dos dados que o usuário insere?

Eles são guardados no seu próprio navegador através de um recurso chamado Local Storage (Armazenamento Local). E mesmo fechando o navegador, eles permanecem no local até o usuário eliminá-los pela opção "Esvaziar a ficha ".

Quais as vantagens de se usar o serviço?

  • Calcula e preenche automaticamente as células correspondentes aos subtotais e totais;
  • É lógico (fornece resultados verdadeiros);
  • Flexibiliza a elaboração do resumo;
  • Salva os dados no navegador do usuário até este eliminá-los.
  • Pode ser impresso.

Política de Armazenamento de dados

Web Storage é um recurso para armazenamento local (no navegador do usuário) de informações por meio de pequenos arquivos de texto, útil para guardar predefinições.

Existem 2 tipos de Web Storage:

  • Session Storage (Armazenamento de Sessão) – elimina os dados gravados quando a janela da página é fechada.
  • Local Storage (Armazenamento Local) – mantém os dados salvos por um período determinado ou até que o usuário exclua.

O Totalizador de Resumo Mensal de Consultas utiliza o primeiro para guardar o consentimento do usuário à Política de Armazenamento e o último pelas razões do subtítulo a seguir.

Por que armazeno dados?

Por 3 razões, descritivamente:

Fazer um backup local
Fazer uma cópia de segurança dos dados da ficha no navegador do usuário. Se forem preenchidos, os dados que são salvos incluem:
  • Os dados da ficha propriamente dita.
  • Dados adicionais como:
    • Nome da Unidade Sanitária;
    • Mês;
    • Nome do Responsável pelo preenchimento;
    • Assinatura do Responsável pelo preenchimento;
    • Data de elaboração;
    • Nota do resumo;
Durabilidade dos dados
Manter salvos os dados previamente inseridos no acto de preenchimento da ficha por tempo suficiente para serem aproveitáveis. Isso é importante, pois possibilita o Profissional interromper sem o risco de voltar à estaca zero, a elaboração do resumo para continuar num outro momento oportuno, sobretudo se ainda lhe faltar dados de alguns indicadores para preencher nas respectivas células da ficha.
Salvar preferências
Por exemplo, quando o usuário marca a opção “Destacar o fundo dos totais" , essa escolha é salva na Web Storage para manter nas suas próximas visitas à plataforma.

Destino dos dados

Todos os dados são salvos localmente (no navegador do usuário). O armazenamento local é para fins de memorização e com isso tornar os dados suficientemente duráveis.

Gerenciamento dos dados pelo usuário

No menu do Totalizador, existe a opção “Esvaziar a ficha” , que permite ao usuário apagar todos os dados da ficha salvos pelo Totalizador na Web Storage.

A aceitação dos termos de armazenamento apresentados neste bloco é dada pela utilização do Totalizador de Resumo Mensal de Consultas.

Como salvar a ficha em PDF?

  1. Clique na opção "Imprimir ".
  2. Na janela que abrirá, no campo de Impressora ou Destino selecione a opção “Salvar como PDF ”.
  3. Por fim, clique em salvar e escolha o destino no seu computador.

Se no navegador Firefox haver omissão das últimas colunas no modo paisagem, teste no modo retrato.
